# Frontend rules

- 1px == 1mm. Не вводить пересчёты и "логические единицы".
- Экспортировать координаты контуров через obj.aCoords.tl.
- Любые расчёты перед экспортом делать при scale=1 (performWithScaleOne).
- API URL для экспорта: POST /api/export-layment.
- Не добавлять тяжёлые фреймворки и сборщики без необходимости.
